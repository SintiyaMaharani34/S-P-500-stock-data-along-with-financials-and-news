# 📈 Stock Market Analysis Project

## 📌 Overview

This project analyzes S&P 500 stock data using:
* Python (Data Processing)
* SQL (Data Analysis)
* Power BI (Visualization)

## 📊 Dataset
* 500+ stocks
* Historical prices
* Financial statements
* News sentiment

## ⚙️ Process
1. Data Cleaning (Python - Google Colab)
2. Data Transformation
3. SQL Analysis
4. Dashboard Visualization

## 🔍 Key Insights
* Identified top-performing stocks
* Analyzed volatility (risk)
* Correlated sentiment with price movement

## 🛠️ Tools Used
* Python
* MySQL
* Power BI

## 📁 Project Structure
* `/data` → dataset
* `/notebooks` → data processing
* `/sql` → analysis queries
* `/dashboard` → visualization
